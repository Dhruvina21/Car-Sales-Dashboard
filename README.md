# 🚗 Car Sales Dashboard – IBM Data Analytics Final Assignment

## 📌 Project Overview
This project analyzes car sales and profits for **SwiftAuto Traders**, a chain of car dealerships. 
The goal is to provide insights on car sales and profits for each dealer.

## 📁 Files Included
| File | Description |
|------|-------------|
| `CarSalesByModelEnd.xlsx` | Excel workbook with 4 chart visualizations |
| `Car_Sales_Dashboard.pdf` | IBM Cognos Analytics dashboard (Sales & Service) |

## 📊 Part 1: Excel Visualizations
- Bar chart – Quantity Sold by Dealer ID
- Line chart – Profit by Date and Model
- Column chart – Profit by Year and Dealer ID
- Line chart – Sum of Profits for Hudson model by Dealer ID

## 📈 Part 2: IBM Cognos Analytics Dashboard

### Sales Dashboard
- KPI: Total Profit ($78.4M)
- KPI: Quantity Sold
- Bar chart: Quantity Sold by Model
- KPI: Average Quantity Sold
- Column chart: Profit by Dealer ID (ascending)

### Service Dashboard
- Column chart: Number of Recalls per Model
- Treemap: Customer Sentiment (Positive/Neutral/Negative)
- Line & Column chart: Quantity Sold vs Profit by Month
- Heat map: Recalls by Model and Affected System

## 🛠 Tools Used
- Microsoft Excel
- IBM Cognos Analytics (Cloud Trial)

Data sourced from the IBM Accelerator Catalog via the Coursera IBM Data Analytics course.
